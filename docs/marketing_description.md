## Directory Consolidator — когда хаос в файлах превращается в порядок

Directory Consolidator — это лёгкая консольная утилита для macOS, которая собирает уникальные файлы из двух исходных папок в одну чистую структуру без дубликатов и пустых директорий. Приложение автоматически сравнивает содержимое по SHA-256, показывает прогресс и лог обработки каждого файла, а при конфликтах имён аккуратно переименовывает копии. Вы просто выбираете три папки через удобный CLI с автодополнением — инструмент берёт на себя контроль точности, очищает целевой путь и даёт готовый консолидированный архив. Подходит для творческих команд, разработчиков и всех, кто хочет быстро навести порядок в разрозненных файловых базах без необходимости писать скрипты вручную.
